# MyUltimateCalendar

MyUltimateCalendar - это мощный и гибкий плагин WordPress для создания и управления календарями на вашем сайте. Плагин предоставляет широкие возможности для отображения событий, расписания и других временных данных в удобном и визуально привлекательном формате.

## Функциональность

- Создание и управление событиями
- Настраиваемые календари с различными стилями отображения
- Поддержка повторяющихся событий
- Интеграция с другими плагинами WordPress
- Возможность настройки цветовой схемы и внешнего вида
- Поддержка нескольких календарей на одном сайте
- Возможность экспорта событий в различные форматы

## Установка и настройка

1. Скачайте плагин MyUltimateCalendar
2. Загрузите файлы плагина в директорию `/wp-content/plugins/` вашего сайта WordPress
3. Активируйте плагин через меню "Плагины" в админ-панели WordPress
4. Перейдите в раздел "MyUltimateCalendar" в админ-панели для настройки параметров календаря
5. Добавьте шорткод `[myultimatecalendar]` на страницу или пост, где вы хотите отобразить календарь

## Часто задаваемые вопросы

### Как создать новое событие?
Для создания нового события перейдите в раздел "MyUltimateCalendar" -> "События" в админ-панели и нажмите "Добавить новое событие".

### Можно ли настроить внешний вид календаря?
Да, вы можете настроить внешний вид календаря через CSS файлы плагина или использовать встроенные опции настройки темы.

### Поддерживает ли плагин импорт/экспорт событий?
Да, плагин поддерживает экспорт событий в форматы iCal и CSV.

### Как добавить календарь на страницу?
Используйте шорткод `[myultimatecalendar]` в редакторе страниц или постов, или виджет календаря в настройках виджетов.
